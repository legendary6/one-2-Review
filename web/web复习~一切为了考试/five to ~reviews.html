字体常用样式

    字体常用样式
  • 字体系列：font-family
      • 设置字体
    • font-family:'宋体';
    • font-family:'宋体'，'仿宋'，'Times New Roman';
    • 若用户系统里有宋体，用宋体，无宋体则用仿宋
  • 字体大小：font-size
      • 字体大小
    • 设置字体大小为绝对值 font-size：20px;
    • 设置字体大小为相对值 font-size: 2em;(或font-size:200%)
    • 相对父元素来设置大小

      • 若未规定字体大小，普通文本的默认大小是16像素

  • 字体风格：font-style
      • 设置字体倾斜显示
    • font-style:italic
    • normal
      italic(italic：浏览器会显示一个斜体的字体样式。)
      oblique(oblique：浏览器会显示一个倾斜的字体样式。)
  • 字体加粗：font-weight
      • 设置字体加粗（固定值）
    • font-weight:900;
    • 属性值取值范围为:100~900
      使用关键字加粗字体
    • font-weight:bolder
    • 属性：lighter normal bold bolder
    • 400等同于normal,而700等同于bold
  • 字体的综合设置
      • font
      • 在一个声明中设置所有属性
      • font:italic bold 36px '宋体'
      • font:36px '宋体'
      • 设置顺序
      • 不强制顺序，但是font-family不能是第一个
      • 未设置的属性值会使用默认值
      • 至少要设置两个属性
  • 使用字体（尽量使用通用字体）
      使用字体
      windows中
    • 中文字体：微软雅黑、黑体、宋体、华文楷体
    • 英文字体：Arial Segone ui
      • 安卓和苹果设备中
    • 中英文：sans-serif
    • 英文字体：Open Sans Helvetica系列

    • 安全字体：Arial,Helvetica,Sans-serif

文本常用样式

水平缩进 text-indent text-indent
    设置段落元素的第一行缩进方式
  • text-indent:2em;
  • text-indent:-3em;
  • 属性值可为绝对值（px）,也可为相对值（em）

    • 实战技巧
  • 常用于设置段落的首航缩进
水平对齐 text-align text-align:文本对齐
    设置元素的文本行对齐方式
  • text-align:left
  • 属性值：left center right
  • 默认对齐方式是左对齐
行高 line-height line-height:行高
    设置行与行之间的距离
  • 固定值（line-height:36px;）
  • 相对值（line-height:1.5em）
文本颜色 color
    设置文本颜色
  • color:green
  • color:#008000
  • color:rgb(0,128.0)
    rgb()
    RGB,将红（red）,绿（green）,蓝（blue）三原色
    的色光以不同的比例相加，以产生多种多样的色光
    rgb(0,0,0)白色 rgb(255,255,255)黑色
文本修饰 text-decoration text-decoration(设置文本修饰效果)
  • none:无装饰
  • overline：上划线
  • underline:下划线
  • line-through：删除线

  • a元素默认有下划线，其他元素默认值为none
  • 可以来去掉浏览器给《a》元素加的默认线
字符转换 text-transform text-transform(设置文本的大小写）~属性值
  • none
  • uppercase:所有的字母转换成大写
  • lowercase：把所有字母转换成小写
  • capitalize：只对每个单词的首字母大写

  • 小细节——行和段落
    行高跟文字要搭配，一般行高是文字的1.5~1.8倍最为合适

    • 为超链接的四种状态设置样式~设置超链接的多种状态时（》=2）时，需要
    按照特定顺序设置

    • :link ~未被访问的超链接
    • :visited ~访问过的超链接
    • :hover ~鼠标经过超链接
    • :active ~链接呗点击的那一刻超链接

    link和visited只能用于超链接

    :focus ~向拥有键盘输入的焦点元素添加样式
    ：first-child ~向元素的第一个子元素添加样式
    :lang ~向带有指定lang属性的元素添加样式

      background-color:1、关键字（gray）

      3、rgb表示方式，如：rgb(128,128,128)
    • 背景图像
      background-image:;
      background-image:url(image/bg.jpg)
    • 背景图像重复
      background-repeat:repeat-x;
      值 描述
      repeat 默认。背景图将在垂直和水平方向重复
      repeat-x 背景图像将在水平方向重复
      repeat-y 背景图像将在垂直方向重复
      no-repeat 背景图像仅显示一次
      inherit 规定应该从父元素继承background-repeat属性设置

    第七章 网页布局

    • 网页布局
    • 网页布局是指网页内容在页面上所处位置的设计
    • 网页布局的宗旨~由整体到局部

    布局历史

    HTML的局限性————块级、行级、行块级
    聪明的方式————table布局
    缺点：工作量大、维护困难 优点：兼容性好
    DIV+CSS
    div-布局标签
    css-布局方法

    布局比较

    布局方法
    浮动——float
      浮动：定义元素在哪个方向浮动，改变页面中对象的前后顺序
    • float:none|left|right
    • 如果包含框无法容纳水平排列的三个浮动元素，那么其他 浮动元素向下移动，直到有了足够的空间
      如果浮动元素的高度不同，浮动块会尽可能向上移动
    • 浮动的影响
      并排显示
      父元素高度塌陷
    • 清除浮动——父元素设置固定高度

      clear:left/right/both/none

      • left:不允许左边有浮动元素
      • right：不允许右边有浮动元素
      • both:不允许左右两边有浮动元素
      • none：默认值，可以浮动

      清除方法~伪元素~:after :before,并搭配属性 clear:both;

    • 溢出处理——overflow
      定义溢出元素内容区如何处理
      值 描述
      visble 默认值。内容不会被修剪，会呈现在元素框之外
      hidden 内容会被修剪，并且其余内容是不可见的
      **scroll 内容会被修剪，但是浏览器会显示滚动条以便查看其余内容
      auto 如果内容被修剪，则浏览器会显示滚动条以便查看其余内容
      inherit 规定应该从父元素继承overflow属性的值

      清除浮动——overflow:hidden;
      ~~父元素不设高度并且子元素浮动时，便父元素高度自动适应子元素高度
      ~~父元素设置的高度或宽度小于子元素时，是父元素出现相应滚动条或隐藏子元素 超出部分内容

        浮动（float）注意事项
      • 浮动元素距离父元素边框的位置，是该侧的父元素padding+
        自身margin的值
      • 通常不会超过父元素的边界
      • 元素一旦浮动起来就不属于父元素
      • 浮动元素不会相互重叠
      • 不能上下浮动，通常只设一种浮动即可
      • 如果父元素宽度不够，浮动元素会令起一行显示
    定位——position

    position定位

    属性 常用值 说明
    position static 默认值。没有定位，元素出现在正常的流中
    relative 相对定位，相对其正常位置进行定位
      相对定位——position：relative;
    • 与left，right，top，bottom等属性共同使用
    • 以自身本应该在的位置为参考物
    • 保留自身位置
    absolute 绝对定位，相对static定位以外的第一个元素有定位 属性的父元素进行定位
      绝对定位——position：absolute;
    • 与left，right，top，bottom等属性共同使用
    • 若祖先元素已定位，则以该祖先元素为参照物
    • 若祖先元素为定位，则以body元素为参照物
    • 不保留自身位置
    fixed 固定定位，相对于浏览器窗口进行定位
      固定定位——position：fixed;
    • 与left，right，top，bottom等属性共同使用
    • 以浏览器窗口进行定位

    float和position不同点
    1、position 与top left等配套使用
    2、float位置移动通过margin \ padding 等实现
    3、布局通常使用float，而不是position

      堆叠顺序————z-index
    • 用于体现元素在Z轴空间的堆叠关系
    • 仅能在定位元素上奏效
    • 大部分元素以数字为取值
    • 可为多个元素设置
