表单：表单是一个包含表单元素的区域。表单元素是允许用户在表单中输入信息的元素。（文本框、 下拉列表、单选框、复选框等等）
其作用是从访问网站的用户那里获得信息，是用用户向服务器传输数据接口

    知识点小结
  • 《form》
    • action:规定向何种发送提交表单数据。值：URL
    • method:规定以何种方式将表单数据传送到服务器。值：get/post

      • 所有表单控件都必须放在 form标签之间，否则用户输入信息无法提交到服务器

    • 文本框
    • 当用户要在表单中输入字母、数字等内容时，就会用到文本框
    • input type="text" name="名称" value="文本"/
      1. type：当type = "text"时，输入框为文本输入框
      2. name：为输入框命名，以备后台程序使用
      3. value：为输入框设置默认值。(一般起到提示作用)
  • 密码框
    • 当用户要在表单中输入密码时，就会用到密码框
    • input type="password" name="名称" value="文本"

    • 表单控件input标签——表单设计调查表是，为了减少 用户的操作，推荐使用选择框。
    HTML中有两种选择框，即单选框和复选框

  • 单选框——radio
    《input type="radio" value="male" name="sex" checked/》男
    《input type="radio" value="female" name="sex"/》女

    同组单选框name属性值必须相同

  • 复选框——checkbox
    《input type="checkbox" value="good1" name="goods"/》 I have a bike 《input type="checkbox" value="good2" name="goods"/》 I have a car

    checked:当添加checked时，该选项默认选中

  • 文件控件——file:当type属性值为file时，用于文件上传
    《input type="file" name="files"/》
    • 按钮
    • 提交按钮：type="submit",提交表单信息到服务器
      《input type="submit" value="提交" name="sub"/》
    • 重置按钮：type="reset",重置表单信息至初始状态
      《input type="reset" value="重置" name="res"/》
    • 普通按钮：type="button"
      《input type="button" value="按钮" name="but"》
  • 多行文本域——《textarea》《/textarea》
      相关属性
    • rows:规定文本区可见行数
    • cols:规定文本区内可见列数
    《textarea name="cat" rows="3" cols="30"》
    The cat was playing in the garden.
    《/textarea》
  • 下拉列表
    《select name="class"》
    《option value="one" selected》软件一班《/option》
    《option value="two" selected》软件二班《/option》
    《option value="three" selected》软件三班《/option》
    《/select》

知识点总结

  • HTML概述——超文本标记语言
    • 超文本：用超链接的方法，将各种不同空间的文字信息组织在一起的网状文本
    • 标记：由标记组成
    • 语言：沟通网页开发者和访问者，由浏览器解析并表达
  • 学习语言的三要素：词汇（标签）、语法（标签的使用规定）、语义（浏览器“理解”的标签含义）
  • 标签分类
    • 双标签
      由“开始标签”和“结束标签”两部分构成，必须成对使用，且必须合理嵌套
      代表标签作用范围
    • 单标签
      在开始标签中进行关闭（以开始标签的结束而结束）
      无需表达范围，仅在标签出现出有效
    • HTML文档结构

      • 相邻兄弟选择器 h1 + p { margin-top : 50px ; }
